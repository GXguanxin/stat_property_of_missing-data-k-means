This is the supplementary material for reproducing all tables and figures of the paper titled
"Statistical Properties of $k$-means Clustering for Data Missing Completely at Random"
using R (version 4.1.3). 

1. The following two files are for defining functions. Conduct them before experiments.
   (1) code_00_cpp_initialcenters_kmmd_once.cpp
   (2) code_01_define_functions.R
2. The rest FIVE .R files are for reproducing Figures 1-4 and Table 1.
